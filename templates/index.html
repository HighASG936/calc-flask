Simple Flask Calculator

Flask microservices that perform basic math operations: addition, subtraction, multiplication, and division.

Endpoints

➕ Addition

GET /adder//

➖ Subtraction

GET /subtractor//

✖️ Multiplication

GET /multiplier//

➗ Division

GET /divider//

Examples

curl https://calc-flask-production.up.railway.app/adder/2/5
2 + 5 = 7.0

curl https://calc-flask-production.up.railway.app/subtractor/10/4
10.0 - 4.0 = 6.0

curl https://calc-flask-production.up.railway.app/multipier/3/7
3.0 * 7.0 = 21.0

curl https://calc-flask-production.up.railway.app/divider/20/4
20.0 / 4.0 = 5.0
